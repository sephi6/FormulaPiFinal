Thank you for supporting us and welcome to UI Pack Toony PRO.

This package contains 300+ textures, 80+ UI prefabs of cartoon style UIs. There are 26 sample scripts and 20 demo scenes.

Special plus, this package comes with GUI Animator for Unity UI, it lets you easily animate all UI objects like a professional. So sweet and worth.

Web demo: http://ge-team.com/pages/unity-3d/ui-pack-toony/

Features:

	• 80+ UI prefabs.
	• 300+ psd files (UHD size).
	• 20 demo scenes for Portrait and Landscape orientations.
	• 26 sample C# scripts.
	• 15 Particle that can work with Unity UI Canvas.
	• 39 sounds.
	• GUI Animator for Unity UI is included.
	• Supports all Unity player platforms.
	• Compatible with Unity 5.0.1 or higher.
