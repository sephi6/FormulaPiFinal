GUI Animator for Unity UI
Version: 0.9.9
- - -- - -- - -- - -- - -- - -- - -- - -- - -- - -- - -- - -- - -- - -

	Quickly and easily add tween animations to Unity UI. 
	This package helps you to reduce working time on UI animations from days/hours to minutes.

	Features:

		• Works with Unity UI. (http://unity3d.com/learn/tutorials/modules/beginner/ui)
		• Works with all Canvas Render Modes.
		• Works with all UI Scale Modes.

		• 10 Demo scenes and 12 sample scripts for C# develoeprs.
		• 9 Demo scenes and 11 sample scripts for JavaScript develoeprs.
		• Animate position, rotation, scale, fade using tweeners.
		• Controls sounds with UI animations.
		• Friendly inspector that categorizes animations into tabs.
		• Ignorable time scale.
		• Callbacks capability.
		
		• Ready to use with DOTween, HOTween, iTween and LeanTween.
		• You can add more tweeners.
		• Use preprocessor directives to examines Tweener before actual compilation.
		
		• Unity Personal and Unity Professional 5.0.1 or higher.
